# Scramble 🥚

A Mac/iOS interactive game showcasing usage of an AirPod in the hand as a game controller.

Hold an AirPod, tilt your hand, catch the eggs.

![scramble in action](demo.gif)

## Build

Open `Scramble.xcodeproj` in Xcode 16+ and run, or:

```sh
xcodebuild -project Scramble.xcodeproj -scheme Scramble -destination 'platform=macOS' build
```
